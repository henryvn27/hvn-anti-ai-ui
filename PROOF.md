# Proof

Anti-AI UI includes working UI examples, not only design notes.

## Demo Surface

- [Tesla sunshade interactive deck](examples/tesla-sunshade-site/index.html)

This example is a single-file product-story UI with:

- asymmetric layout
- product-specific copy
- slide navigation
- priority cards
- buyer persona cards
- final recommendation state
- non-default palette and type rhythm

It is intentionally not the generic AI pattern of centered hero, purple gradient, three equal cards, icon circles, and vague SaaS copy.

## Verification

```sh
npm run lint
npm run build
```

The docs site and library build from source. The example can be opened directly from `examples/tesla-sunshade-site/index.html`.
