# Engineering Feedback

Matt Pocock Skills are not a UI system, but two rules apply directly to anti-AI UI work:

- Use a tight visual loop before judging a page: run the docs site, inspect the actual screen, then fix the specific failure.
- Use shared language in copy and components: name real product concepts instead of generic nouns like platform, workflow, solution, or experience.

For implementation work, route through ORCA or HVN Stack. This repo only keeps the UI-facing discipline.
