- slide-by-slide deck navigation
- progress rail
- priority cards
- buyer persona cards
- final weighted recommendation chooser
- hidden Bailey mode / Charleston branch microcopy layer

## Vercel deploy notes

This example is a static single-file site.

To deploy as its own Vercel project from GitHub:

1. Import `henryvn27/hvn-anti-ai-ui` into Vercel.
2. Set the project root directory to `examples/tesla-sunshade-site`.
3. Use no framework preset or a static HTML preset.
4. No build command is required.

If you want it promoted from example/demo to a standalone production repo, create a dedicated repo and move this folder to the root.
